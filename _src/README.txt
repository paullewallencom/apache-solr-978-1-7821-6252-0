"Apache Solr's Beginners Guide" examples.

This is the repository containing the examples for the book:
https://www.packtpub.com/apache-solr-beginners-guide/book


The book provides a step-by-step introduction to Solr, using essential examples.
The examples focus on simple tasks and the text is also designed to give to the reader 
a wide overview of the context where this technology lives.
